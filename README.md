# GitHired
Job search via programming language or location via GitHub Jobs API

https://fathomless-earth-60439.herokuapp.com/

### Prerequisites

Install project dependencies:

```
rbenv install 2.5.0
```

```
brew install postgres
```

Setup the development environment:

```
$ bin/rails db:setup
```

Run the specs:

```
$ bin/rspec spec
```

## Getting Started

Give it a go!

```
$ bin/rails s
```
